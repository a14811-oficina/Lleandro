
# Exercícios de Python – Módulo 3  
**Programação de Sistemas Informáticos**

Este repositório contém os exercícios desenvolvidos em Python no âmbito do **Módulo 3** da disciplina **Programação de Sistemas Informáticos**, do **Curso Profissional Técnico de Gestão e Programação de Sistemas Informáticos**.

Os exercícios abordam conceitos fundamentais da programação em Python, desde instruções básicas até estruturas de controlo, ciclos, listas e funções.

---

## 📚 Conteúdos Abordados

### 🔹 Introdução ao Python
- Utilização da função `print()`
- Declaração e uso de variáveis
- Tipos de dados (`int`, `float`, `str`)
- Entrada de dados com `input()`

---

### 🔹 Operações Matemáticas
- Soma, subtração, multiplicação e divisão
- Conversão de temperaturas (Celsius ↔ Fahrenheit)
- Cálculo da idade aproximada
- Operadores aritméticos e lógicos

---

### 🔹 Estruturas Condicionais (`if`, `elif`, `else`)
- Verificação de notas (aprovado/reprovado)
- Classificação por idade (criança, jovem, adulto, sénior)
- Verificação de números pares e ímpares
- Comparação de valores (maior número)
- Autenticação simples (utilizador e palavra-passe)
- Cálculo de descontos
- Verificação de múltiplos (3 e 5)
- Intervalos de valores
- Ano bissexto
- Cálculo do IMC (Índice de Massa Corporal)

---

### 🔹 Ciclos de Repetição
#### `while`
- Contadores crescentes e decrescentes
- Tabuadas
- Menus interativos
- Programas em ciclo contínuo até condição de saída

#### `for`
- Listagem de números
- Números pares
- Soma de sequências
- Cálculo de fatoriais
- Iteração com `range()`

---

### 🔹 Listas e Estruturas de Dados
- Criação e manipulação de listas
- Inserção de notas
- Listagem de valores
- Cálculo de médias
- Filtragem de valores positivos

---

### 🔹 Funções
- Definição e utilização de funções
- Organização do código em módulos
- Passagem de parâmetros

---

### 🔹 Projeto Final – Gestão de Stocks
Sistema completo em Python com:
- Registo de novos materiais
- Consulta de stock por material
- Atualização de quantidades (adicionar/remover)
- Visualização geral do stock
- Menu interativo em consola

---

## 🛠️ Tecnologias Utilizadas
- **Python 3**
- Ambiente de consola/terminal

---

## 🎯 Objetivos de Aprendizagem
- Compreender a sintaxe básica de Python
- Desenvolver raciocínio lógico
- Aplicar estruturas de decisão e repetição
- Trabalhar com listas e funções
- Criar programas interativos em consola
